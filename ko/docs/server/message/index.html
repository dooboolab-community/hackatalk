메시지

메시지를 보내려는 채널의 고유 id를 알고 있으면 메시지를 간단하게 보낼 수 있습니다.

메시지 종류#

MessageType에 메시지 종류들이 정의되어 있습니다.

  1. text
    • 텍스트 메시지
  2. photo
    • 이미지를 바로 표시하는 메시지
  3. file
    • 다운로드 링크를 가지는 메시지

photo와 file은 URL이 포함 된 문자열 배열로 제공됩니다. URL은 singleUpload 쿼리에 의해 제공됩니다. 미래에는 createMessage에 업로드 부분도 통합되어야 할 것입니다.

Push Notification#

모든 장치와 브라우저에는 푸시 토큰 ID가 있습니다. 이는 기기에서 푸시 알림을 수신할 때 사용됩니다. 사용자가 로그인하면 단말의 푸시 토큰이 서버로 전송됩니다. createNotification 쿼리가 이를 담당합니다. 사용자의 고유 id와 기기의 푸시 토큰은 데이터베이스에서 매핑되며 푸시 알림을 통해 원하는 단말로 메시지를 보내는 데 사용됩니다. 사용자가 로그아웃 할 때deleteNotification 쿼리를 호출해서 매핑을 삭제하면 잘못된 대화가 전송되지 않을 것입니다.
